Oráculo (des)Motivador

Projeto simples desenvolvido como exercício prático de aprendizado em **JavaScript + Node.js + Express**.

A proposta do site é simular um "oráculo pessimista" que retorna frases irônicas e desmotivadoras sobre três áreas da vida:

* 💼 Trabalho
* ❤️ Vida amorosa
* 🧠 Vida pessoal

Cada clique retorna uma frase aleatória diferente armazenada em um banco local (JSON).

---

## Objetivo do projeto

Este projeto foi criado **exclusivamente para fins de estudo e prática**, com foco em:

* manipulação de eventos no frontend
* comunicação frontend ↔ backend
* criação de API simples com Express
* leitura de arquivos JSON como banco local
* organização básica de estrutura de projeto web
* versionamento com Git
* publicação online

---
